# BGmi RTGTX7 定制版 1.1.0

发布日期：2026-04-20

这是基于上一版 **1.0** 继续迭代的 **1.1** 版本，本次不替换旧版发布，只新增一个新版本。

## 版本重点

### Bangumi 季度分组

- Bangumi 首页改为按季度分组展示
- 季度来源复用现有 Mikan 海报日期来源与封面缓存逻辑
- 新增季度详情页
- 支持从首页分组直接进入对应季度完整列表

### Bangumi 布局优化

- 修复桌面端季度分组在窗口尺寸变化后不会自动重排的问题
- 修复季度分组只有 1 个番剧时海报被拉伸的问题
- 优化移动端季度详情页为更合理的双列海报布局
- 修复移动端季度详情页海报过小、左右留白过大的问题

### 播放器移动端体验

- 修复移动端 Chrome 长按视频时被浏览器原生菜单截胡的问题
- 保留并恢复自定义长按 2 倍速交互

### 局域网调试

- 启用前端 Vite 局域网访问，方便直接用手机测试移动端页面与播放器

## 适合谁更新

如果你当前主要在以下场景使用 BGmi，这个版本建议更新：

- 需要按季度浏览 Bangumi
- 经常在手机上查看番剧列表
- 需要在手机浏览器中测试播放器交互
- 希望首页季度分组在桌面端更规整、更接近可用状态

## 已验证

- 前端构建通过：`corepack pnpm build`
- 后端测试通过：`pytest BGmi/tests/test_utils.py BGmi/tests/test_http_api.py`
