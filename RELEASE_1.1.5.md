# BGmi RTGTX7 定制版 1.1.5

发布日期：2026-04-20

这是在 **1.1.0** 基础上的持续迭代版本，主要聚焦在 **Bangumi / Archive 页面结构优化、Subscribe Dashboard 管理面板、以及桌面端海报布局自适应**。

## 本次更新重点

### 1. Bangumi / Archive 页面拆分与整理

- `Bangumi` 页面现在只保留 **当季新番**
- 历史季度内容迁移到 **Archive / 往期番剧**
- 支持从季度分组进入完整季度详情页
- 去掉了多余的排序切换和冗余副标题，让页面更干净

### 2. 桌面端海报布局优化

- 优化了 `Bangumi` 与 `Archive` 的桌面端主内容区域宽度
- 海报网格现在会根据可用空间自适应排布
- 减少右侧大面积留白，让海报尽量占满内容区
- 同步优化：
  - Bangumi 当季页
  - Archive 首页季度分组预览
  - Archive 季度详情页

### 3. Subscribe Dashboard 管理面板

- 在 `Subscribe` 页面中新增 `Dashboard` 标签页
- 提供数据总览、异常诊断和维护操作入口
- 新增维护操作：
  - 重新同步 Mikan 数据
  - 检查异常数据
  - 重建仓库番剧
  - 剧集全部清零
  - 提交下载任务
  - 更新剧集和海报
- 高风险操作继续保留确认 / double check 逻辑

### 4. Dashboard UI 与可读性

- 优化了 Dashboard 的 sci-fi / HUD 风格
- 修复深色模式下文字过暗、按钮不可读的问题
- 修复亮色模式下 `SYNC OK / READY / ALL SYSTEMS NORMAL` 等状态 chip 对比度不足的问题

### 5. 移动端与交互细节

- 优化 Subscribe / Calendar 的移动端星期切换交互
- 优化 Bangumi 移动端标题区域与入口层级
- 优化订阅卡片和详情弹层体验
- 修复移动端播放器点击 / 长按手势冲突：轻点可正常呼出控制栏，长按仍保持 2x 倍速，并继续屏蔽 Chrome 原生长按下载菜单

## 适合更新的人群

如果你现在主要关注以下场景，建议升级到 **1.1.5**：

- 想把当季新番和历史季度分开浏览
- 希望桌面端海报列表更规整、更铺满
- 需要一个可直接执行维护操作的 Subscribe Dashboard
- 经常在手机上查看 Bangumi / Subscribe / Calendar
